comandos git 

1. Seção: Sequência Introdutória 
Intro1 
Objetivo: Criar um commit no repositório. 
Comando utilizado: 
git commit 

Intro2 
Objetivo: Criar uma nova branch chamada bugFix. 
Comando utilizado: 
git branch bugFix 

Intro3 
Objetivo: Alternar para a branch criada. 
Comando utilizado: 
git checkout bugFix 

Intro4 
Objetivo: Criar uma branch, realizar commits em diferentes branches e alternar entre 
elas. 
Comandos utilizados: 
git checkout -b bugFix 
git commit 
git checkout main 
git commit 

2. Seção: Acelerando (Ramp Up) 
Rampup1 
Objetivo: Realizar um merge entre branches. 
Comandos utilizados: 
git checkout bugFix 
git commit 
git checkout main 
git merge bugFix 

Rampup2 
Objetivo: Utilizar o comando rebase para reorganizar o histórico de commits. 
Comandos utilizados: 
git checkout bugFix 
git commit 
git checkout main 
git commit 
git rebase main 

Rampup3 
Objetivo: Atualizar uma branch utilizando rebase a partir da branch principal. 
Comandos utilizados: 
git checkout bugFix 
git rebase main 

Rampup4 
Objetivo: Atualizar a branch principal utilizando rebase a partir da branch bugFix. 
Comandos utilizados: 
git checkout main 
git rebase bugFix 

3. Seção: Repositórios Remotos 
Remote1 
Objetivo: Enviar alterações para o repositório remoto. 
Comando utilizado: 
git push 

Remote2 
Objetivo: Buscar alterações do repositório remoto. 
Comando utilizado: 
git fetch 

Remote3 
Objetivo: Atualizar o repositório local com as mudanças do remoto. 
Comando utilizado: 
git pull 

Remote4 
Objetivo: Enviar uma branch específica para o repositório remoto. 
Comando utilizado: 
git push origin main 

Remote5 
Objetivo: Criar ou enviar uma branch chamada bugFix para o repositório remoto. 
Comando utilizado: 
git push origin bugFix 

Remote6 
Objetivo: Buscar alterações do repositório remoto e integrar com a branch local. 
Comandos utilizados: 
git fetch 
git merge origin/main 

Remote7 
Objetivo: Atualizar a branch local utilizando rebase com base na branch remota. 
Comandos utilizados: 
git fetch 
git rebase origin/main
